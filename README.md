Student Portal 
A full-stack web application for student registration, login, and profile management — built with Node.js, Express, and vanilla JavaScript.

Live Demo
View Live on Render 

Features
Student Registration with form validation
Secure Login system
Profile view and update
Dashboard after successful login
Data stored in JSON 
Fully responsive design

Tech Stack

Frontend
HTML, CSS, JavaScript

Backend
Node.js, Express.js

Storage
JSON file (users.json)

Deployment
Render

Version Control
Git & GitHub
